# OSINT
An array of OSINT tools in python3.


How to use:

1) Navigate to the folder where install will stay:
   a) /tmp folder to use temporaly 
   b) /opt folder to keep it permanently
   c) /usr/local/bin/ to use it as a command in $PATH


2) Download the master branch using git clone:

   git clone --branch master https://github.com/Hyperborea284/OSINT.git 


3) Create a venv in the folder:
   https://python.land/virtual-environments/virtualenv
    
    cd OSINT/
 
    python -m venv .


4) Install the requirements.txt

    pip3 install -r requirements.txt
    
    
5) Use with the command; Insert the link for an youtube video:
    
    python3 main.py


6) The results will apperar in the scores/ folder.
